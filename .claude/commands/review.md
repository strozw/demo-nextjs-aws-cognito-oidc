# コードレビュー

AGENTS.md に記載された開発ガイドラインに基づいて、以下の観点でコードレビューを実施してください:

## チェック項目

### TypeScript
- [ ] strict モードに準拠しているか
- [ ] any 型を使用していないか
- [ ] 適切な型定義がされているか

### Next.js v16 / React v19.2
- [ ] Server Components と Client Components が適切に使い分けられているか
- [ ] "use client" ディレクティブが必要な箇所のみに使用されているか
- [ ] React Compiler の恩恵を受けられる書き方になっているか

### セキュリティ
- [ ] 環境変数から機密情報を読み込んでいるか
- [ ] クライアントサイドに機密情報が露出していないか
- [ ] CSRF/XSS 対策が実装されているか
- [ ] 認証・認可が適切に実装されているか

### コード品質
- [ ] ESLint v9 のルールに違反していないか
- [ ] Prettier でフォーマットされているか
- [ ] shadcn/ui コンポーネントを優先的に使用しているか

### パフォーマンス
- [ ] 不要な useState や useEffect を使用していないか
- [ ] Server Components でデータフェッチを行っているか

レビュー結果と改善提案を提示してください。
